## Descripción del taller

Este taller contiene ejemplos de funciones recursivas en Java. Incluye implementaciones de algoritmos clásicos como factorial, Fibonacci, máximo común divisor, suma de vectores y más, con el objetivo de practicar la recursividad y la resolución de problemas.

## Instrucciones para ejecutar el programa

1. Abre una terminal en el directorio del proyecto.
2. Compila las clases Java del directorio `src`:

```bash
javac src/*.java
```

3. Ejecuta la clase deseada usando el paquete `src`. Por ejemplo, para ejecutar `PotenciaRecursiva`:

```bash
java -cp src src.PotenciaRecursiva
```

4. Si tu clase principal usa entrada de usuario o un método específico, ajusta el comando anterior según corresponda.
